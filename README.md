## 如何打包

```json
cd mind-map/web && npm run build
cd ../.. && npm run build
```

## 致谢

- 基于[mindmap](https://github.com/wanglin2/mind-map): 
  - 魔改部分：
    - 支持大纲复制为Markdown多级列表和粘贴多级列表
    - 优化保存图片：节点高亮和折叠按钮隐藏
    - 节点也可以直接选中粘贴多级列表批量粘贴出多个节点
