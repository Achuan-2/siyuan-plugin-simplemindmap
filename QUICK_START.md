# Quick Start Guide - 思维导图插件

## 快速开始

### 1. 创建思维导图

在思源笔记中输入斜杠命令:
```
/mindmap
```
或
```
/simple-mind-map
```

插件会自动:
- 创建一个占位图片
- 初始化空的思维导图数据
- 打开编辑器

### 2. 编辑思维导图

#### 方式一: 从斜杠命令创建后直接编辑
创建后会自动打开编辑器

#### 方式二: 右键菜单编辑
1. 右键点击思维导图图片
2. 选择 "Edit Mind Map"

### 3. 保存思维导图

在 simple-mind-map 编辑器中:
1. 编辑思维导图 (添加节点、修改文本等)
2. 编辑器会自动保存数据到块属性
3. 保存后会自动导出为图片并更新显示

### 4. 查看思维导图

#### 方式一: 在文档中查看
思维导图会以图片形式显示在文档中

#### 方式二: 右键菜单查看
1. 右键点击思维导图图片
2. 选择 "View Mind Map"

## 编辑器操作

### 基本操作

- **添加子节点**: 选中节点后按 Tab 键
- **添加同级节点**: 选中节点后按 Enter 键
- **删除节点**: 选中节点后按 Delete 键
- **编辑节点**: 双击节点
- **移动节点**: 拖拽节点
- **缩放画布**: 鼠标滚轮
- **平移画布**: 按住空白区域拖拽

### 工具栏功能

simple-mind-map 提供了丰富的工具栏功能:
- 节点样式设置
- 主题切换
- 布局切换
- 导出功能
- 等等...

## 配置选项

在插件设置中可以配置:

### 1. 标签显示 (labelDisplay)
- **不显示**: 不显示任何标签
- **始终显示**: 始终显示标签
- **悬停显示**: 鼠标悬停时显示标签

### 2. 嵌入图片格式 (embedImageFormat)
- **SVG**: 矢量图,文件小,可无限缩放
- **PNG**: 位图,兼容性好

### 3. 编辑窗口 (editWindow)
- **Dialog**: 在对话框中编辑
- **Tab**: 在新标签页中编辑 (桌面端推荐)

### 4. 主题模式 (themeMode)
- **浅色**: 使用浅色主题
- **深色**: 使用深色主题
- **跟随系统**: 跟随思源笔记主题

## 数据存储

### 思维导图数据
- 存储位置: 块属性 `custom-mindmap`
- 格式: JSON 字符串
- 包含: 节点数据、主题、布局等

### 图片文件
- 存储位置: `data/assets/mindmap-image-*.{png|svg}`
- 格式: PNG 或 SVG
- 自动更新: 保存时自动导出并更新

## 常见问题

### Q: 思维导图数据会丢失吗?
A: 不会。数据保存在块属性中,与文档一起同步。

### Q: 可以导出思维导图吗?
A: 可以。思维导图已经导出为图片,可以直接使用。也可以在编辑器中使用导出功能导出其他格式。

### Q: 支持哪些格式?
A: 目前支持 PNG 和 SVG 两种图片格式。

### Q: 可以在移动端使用吗?
A: 可以,但推荐在桌面端使用以获得更好的体验。

### Q: 编辑器语言可以切换吗?
A: 编辑器语言会自动跟随思源笔记的语言设置。
